GreenSock

What is GreenSock? (GSAP)

The Greensock Animation Platform (GSAP) is a standard Javascript library of small Javascript files used for animating elements and SVGs. It is highly performant and compatible with every major browser.Greensock has been around for over ten years and is one of the most popular animation platforms out there.

How does it work ?

All you have to do is write short snippets of Javascript that define how elements should animate and set the timing of the animations. Start by installing one of four libraries before your Javascript file.

\

GSAP AND SVGS ?

it is a powerful tool for animating SVGs. SVG animation is often buggy and Greensock clears these bugs. Features include the ability to animate SVG attributes, progressively reveal strokes, morph SVG shapes, drag and drop SVG elements, and more
